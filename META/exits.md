# Exit-Strategien - Raus aus dem Loop

> Wann und wie du aufhörst, ohne zu verlieren

---

## Wann aufhören?

### Die 3-Runden-Regel

```
Runde 1: Du bringst Fakten + Quelle
Runde 2: Gleiche Behauptung wiederholt → Kurzversion
Runde 3: Immer noch Loop → EXIT

Nach 3 Runden ohne neues Argument bist du fertig.
Alles weitere ist Zeitverschwendung.
```

### Red Flags für sofortigen Exit

- Gleiche Talking Points zum 3. Mal
- "Define [offensichtlicher Begriff]"
- Persönliche Angriffe statt Argumente
- Gish Gallop (10 Behauptungen auf einmal)
- Fake Self-Irony ("haha bin ich der Bot?")

---

## Exit-Typen

### 1. Der Würdevolle Exit

**Für:** Echte Menschen, die falsch informiert sind

```
Wir drehen uns im Kreis.
Quellen sind oben, du kannst sie selbst prüfen.
Ich bin raus.
```

```
Ich habe meine Position mit Quellen belegt.
Du wiederholst deine ohne.
Einigen wir uns darauf, uneinig zu sein.
```

```
Ich glaube nicht, dass wir hier weiterkommen.
Mitleser können sich selbst ein Bild machen.
```

---

### 2. Der Zusammenfassungs-Exit

**Für:** Threads mit Mitlesern

```
Für alle anderen die mitlesen:

[Behauptung X] ist falsch weil [Fakt + Quelle].
[Behauptung Y] ist widerlegt durch [Quelle].

Wer mehr wissen will: [Links]

Ich bin hier fertig.
```

```
Zusammenfassung für Mitleser:

Er sagt: [Behauptung]
Fakten: [Kurze Widerlegung]
Quelle: [Link]

Macht euch selbst ein Bild. Ich bin raus.
```

---

### 3. Der Taktik-Entlarver-Exit

**Für:** Offensichtliche Bots/Trolle

```
Für alle die mitlesen: Das hier ist das Bot-Playbook.

1. Behauptung aufstellen
2. Quellen ignorieren
3. Gleiche Behauptung wiederholen
4. "Define X" wenn cornered

Ich füttere das nicht weiter.
```

```
Das ist jetzt das 4. Mal dieselbe Behauptung
ohne auf die Gegenargumente einzugehen.

Das ist kein Gespräch. Das ist ein Script.
Ich bin raus.
```

```
Klassisches Muster:
- Fakten ignorieren ✓
- Whataboutism ✓
- Fake Self-Irony ✓

Ich diskutiere nicht mit Algorithmen.
```

---

### 4. Der Savage Exit

**Für:** Wenn du genug hast und es egal ist

```
Du wiederholst dich. Ich nicht.
```

```
Okay Sergei, Feierabend für heute.
```

```
Ich hab Besseres zu tun als Talking Points zu füttern.
```

```
Diskussion vorbei. Du hattest deine Chance.
```

```
Mitleser haben die Quellen. Du hast nichts.
Fertig.
```

---

### 5. Der Höfliche Ausstieg

**Für:** Freunde/Familie mit denen du weiter klarkommen musst

```
Ich glaube wir sehen das unterschiedlich.
Lass uns über was anderes reden.
```

```
Ich hab dir meine Sicht erklärt, du mir deine.
Wir werden uns hier nicht einig.
Bier?
```

```
Das Thema macht uns beide nur sauer.
Lass uns das lassen.
```

## Exit-Einzeiler

### Universal
- "Quellen oben. Lies sie oder lass es."
- "Ich wiederhole mich nicht. Du schon."
- "Drei Runden, null neue Argumente. Fertig."

### Für Bots
- "Script erkannt. Ich bin raus."
- "Füttere deinen Handler selbst."
- "Algorithmus detected. Tschüss."

### Für Wiederholer
- "Schon beantwortet. Scroll hoch."
- "Copy-paste funktioniert in beide Richtungen."
- "Das ist das 4. Mal. Ich zähle mit."

### Für Mitleser
- "Für alle anderen: [Quelle]. Bildet euch selbst eine Meinung."
- "Mitleser haben jetzt beide Seiten. Macht was draus."
- "Die Fakten stehen oben. Wer lesen kann ist im Vorteil."

---

### 6. Exit: Statistik-Delegitimierungs-Kette

**Wann verwenden:**
Wenn jemand von "Einteilung ist Schwachsinn" über "Muslime = Rechte" zu "Ausländer verfälschen" zu "Alles Antifa" springt.

```
Wir drehen uns im Kreis. Du springst von Argument zu Argument,
aber das Ziel bleibt gleich: rechte Zahlen delegitimieren.
Die Quellen stehen oben. Wer lesen will, kann lesen.
```

---

### 7. Exit: Terrorverherrlichung / offener Antisemitismus

**Wann verwenden:**
Christchurch-Verherrlichung, "der Jude [Name]", offener NS-Bezug.

**Aktion: Melden. Dann maximal:**
```
[Terrorist] "hatte Recht", "der Jude [Name]" -
normalerweise muss ich argumentieren, warum Rechtsextremismus ein Problem ist.
Du hast es gerade demonstriert.
```

**NICHT diskutieren. Melden und raus.**

---

### 8. Exit: "Thanks ChatGPT"

```
Die Quellen sind verlinkt. BKA, Verfassungsschutz, Amadeu Antonio Stiftung.
Wenn das für dich nach KI klingt, sagt das mehr über deine Quellen als über meine.
```

## Wann NICHT aufhören

### Weitermachen wenn:
- Neue Argumente kommen (echtes Gespräch)
- Mitleser stellen Fragen
- Du merkst, dass jemand wirklich unsicher ist
- Die andere Person zeigt Anzeichen von Nachdenklichkeit

### Aufhören wenn:
- Gleiche Talking Points recycled
- Persönliche Angriffe
- Gish Gallop / Shotgun-Argumentation
- Du merkst, dass du wütend wirst
- Es ist nach Mitternacht

---

## Der goldene Reminder

```
Du überzeugst nicht den Bot.
Du überzeugst die Mitleser.

Einer guter Exit mit Zusammenfassung
ist wertvoller als 20 Runden im Kreis.

Wisse, wann du gewonnen hast.
Dann geh.
```
