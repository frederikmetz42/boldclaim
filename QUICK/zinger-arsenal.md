## Deflection-Konter

### "Das hat er nie gesagt!"
```
Doch. Hier ist das Video. Hier ist das Gerichtsurteil.
Du leugnest jetzt Rechtstatsachen.
```

### "Aus dem Kontext gerissen!"
```
Okay. Welcher Kontext macht [X] akzeptabel?
Ich warte.
```

### "Das ist deine Interpretation!"
```
Nein. Das ist ein Gerichtsurteil.
OVG Münster. Er hat geklagt. Er hat verloren.
```

### "Immer kurz vor Wahlen!"
```
Sachsen-Anhalt: Einstufung NACH der Wahl.
Thüringen: 3 Jahre VOR der Wahl.
5 Jahre Prüfzeit.

Deine Verschwörungstheorie hat ein Timing-Problem.
```

### "Die anderen machen das auch!"
```
Zwei Unrechte ergeben kein Recht.
Nenn mir konkrete Beispiele - dann reden wir drüber.
Bis dahin: Whataboutism ist kein Argument.
```

### "Wo sind die Beweise?"
```
Ermittlungen laufen. Due process braucht Zeit.
"Noch nicht verurteilt" ≠ unschuldig.
Oder willst du Schnellgerichte ohne Beweisaufnahme?
```

## Rhetorik-Fallen

### "Ich bin ja kein Nazi, ABER..."
```
Alles vor dem "aber" ist irrelevant.
Was kommt nach dem "aber"?
```

### "Man wird ja wohl noch sagen dürfen..."
```
Du darfst alles sagen.
Und ich darf dir sagen, warum es Bullshit ist.
Meinungsfreiheit ist keine Kritikfreiheit.
```

### "Das wird man ja wohl fragen dürfen..."
```
Du fragst nicht. Du insinuierst.
Stell eine echte Frage oder lass es.
```

### "Ich bin nur besorgt..."
```
Echte Sorge führt zu Lösungsvorschlägen.
Du hast nur Feindbilder.
```

### "Die Wahrheit liegt in der Mitte..."
```
Die Erde ist keine Scheibe.
Die Wahrheit liegt nicht zwischen Fakt und Lüge.
```

### "Alle Experten/Medien/Gerichte liegen falsch..."
```
700 Verfassungsschützer. Unabhängige Richter. Investigativjournalisten.
Alle in einer Verschwörung?

Oder du liegst falsch.
Occam's Razor.
```
