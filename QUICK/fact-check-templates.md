## Die Anatomie einer Lüge

### Wie Desinformation funktioniert:

**1. Kern-Wahrheit verdrehen:**
- Nimm eine echte Statistik
- Entferne Kontext
- Präsentiere verzerrtes Bild

**Beispiel:**
- Wahrheit: "2017 stiegen Sexualdelikte"
- Fehlt: Gesetzesänderung 2016 erfasst mehr Taten
- Lüge: "Flüchtlinge verursachen Vergewaltigungswelle"

**2. Autoritäten missbrauchen:**
- "Studie zeigt..." (welche Studie?)
- "Experten sagen..." (welche Experten?)
- "Wissenschaftlich bewiesen..." (peer-reviewed?)

**3. Emotionen triggern:**
- Angst: "Unsere Kinder sind in Gefahr!"
- Wut: "Die belügen uns!"
- Ekel: "Die sind schmutzig/kriminell"

## Statistik-Manipulation erkennen

### Die 5 häufigsten Tricks:

**1. Cherry-Picking:**
- Nur Daten zeigen, die zur These passen
- Rest ignorieren

**Check:** Gibt es andere Zeiträume/Regionen mit anderen Ergebnissen?

**2. Basis-Verschiebung:**
- "100% Anstieg!" (von 2 auf 4 Fälle)
- Absolute vs. relative Zahlen

**Check:** Was sind die absoluten Zahlen?

**3. Korrelation ≠ Kausalität:**
- "X passiert gleichzeitig mit Y, also verursacht X Y"
- Eisverkauf und Ertrinken steigen beide im Sommer

**Check:** Gibt es eine dritte Variable?

**4. Zeitliche Tricks:**
- Gesetzesänderungen ignorieren
- Erfassungsmethoden ändern sich

**Check:** Was hat sich im Erfassungszeitraum geändert?

**5. Apfel-Birnen-Vergleiche:**
- Unterschiedliche Definitionen
- Unterschiedliche Messmethoden

**Check:** Werden gleiche Dinge verglichen?

---

### Statistik-Check Template

```
Behauptung: "[X] zeigt, dass [Y]"

□ Quelle genannt?
□ Primärquelle zugänglich?
□ Zeitraum angegeben?
□ Absolute Zahlen verfügbar?
□ Kontext erklärt?
□ Vergleichsdaten vorhanden?
□ Erfassungsmethode erklärt?
□ Mögliche Confounders genannt?

Fehlende Checks = Misstrauen angebracht
```

## Quellen-Bewertung

### Quellen-Hierarchie (Vertrauenswürdig → Weniger)

**Tier 1 - Primärquellen:**
- Gerichtsurteile (OpenJur)
- Gesetzestexte (Bundestag)
- Offizielle Statistiken (Destatis, BKA, BAMF)
- UN-Dokumente
- Peer-reviewed Studien

**Tier 2 - Qualitätsjournalismus:**
- Investigativ (Correctiv, Bellingcat)
- Öffentlich-rechtlich (Tagesschau, ZDF)
- Qualitätspresse (Zeit, SZ, Spiegel)
- Faktenchecker (Tagesschau Faktenfinder, AFP)

**Tier 3 - Sekundärquellen:**
- Think Tanks (mit Bias-Check)
- Fachblogs
- Lokale Medien

**Tier 4 - Mit Vorsicht:**
- Boulevardpresse
- Partei-nahe Medien
- Social Media Posts

**Tier 5 - Meist unzuverlässig:**
- Anonyme Blogs
- Telegram-Kanäle
- "Alternative Medien" ohne Impressum

---

### Quellen-Red-Flags:

```
□ Kein Impressum
□ Keine Autorennennung
□ Keine Quellenangaben
□ Reißerische Headlines
□ Ausschließlich emotionale Sprache
□ Werbung für "geheimes Wissen"
□ "Mainstream-Medien lügen"
□ Keine Korrektur-Politik
```

## Debunking-Templates

### Template 1: Statistik-Debunk

```
**Behauptung:** "[Statistik X zeigt Y]"

**Was fehlt:**
1. [Fehlender Kontext 1]
2. [Fehlender Kontext 2]

**Die vollständige Geschichte:**
- [Zeitlicher Kontext]
- [Methodischer Kontext]
- [Was wirklich passiert ist]

**Quelle:** [Primärquelle]

**TL;DR:** [Kurze Zusammenfassung]
```

**Beispiel ausgefüllt:**
```
**Behauptung:** "2017 explodierten Sexualdelikte wegen Flüchtlingen"

**Was fehlt:**
1. 2016 wurde das Sexualstrafrecht reformiert ("Nein heißt Nein")
2. Neue Straftatbestände wurden eingeführt

**Die vollständige Geschichte:**
- Flüchtlinge kamen 2015
- Gesetzesänderung 2016 erfasst mehr Taten
- Statistik steigt 2017

**Quelle:** Bundestag 2016 - Sexualstrafrecht

**TL;DR:** Die Statistik steigt, weil mehr erfasst wird - nicht weil mehr passiert.
```

### Template 2: Zitat-Debunk

```
**Behauptetes Zitat:** "[Zitat]"

**Faktencheck:**
□ Originalquelle gefunden: [Ja/Nein]
□ Kontext vorhanden: [Ja/Nein]
□ Datum verifiziert: [Ja/Nein]

**Ergebnis:**
- [Wahr / Teilwahr / Falsch / Aus Kontext]
- [Was wirklich gesagt wurde]
- [Vollständiger Kontext]

**Quelle:** [Original]
```

3. [Grund 3]

**Die Fakten:**

**Ursprung der Falschinformation:**
[Woher kommt die Behauptung? Wer profitiert?]

## Red Flags checken

### Nachrichten-Check:

```
□ Wird nur EINE Quelle genannt?
□ Sind alle Zitate anonym?
□ Fehlen gegenteilige Stimmen?
□ Ist die Headline sensationalistischer als der Inhalt?
□ Werden Emotionen stärker angesprochen als Fakten?
□ Wird "Mainstream" als Feind dargestellt?
□ Gibt es einen Call-to-Action (Teilen, Petitionen)?
```

### Social-Media-Post-Check:

```
□ Account-Alter unter 6 Monaten?
□ Nur politische Posts?
□ Keine persönlichen Inhalte?
□ Extreme Posting-Frequenz?
□ Antworten auf viele verschiedene Threads?
□ Wiederholt gleiche Talking Points?
□ Ignoriert Gegenargumente komplett?
```

### Video-Check:

```
□ Quelle des Clips angegeben?
□ Datum verifizierbar?
□ Vollständiger Kontext verfügbar?
□ Schnitte auffällig?
□ Audio/Video synchron?
□ Umgekehrte Bildersuche möglich?
```

## Response-Frameworks

### Framework 1: Kurz und Sachlich

```
Falsch. [Fakt] + [Quelle]
```

**Beispiel:**
"Falsch. Höcke darf laut OVG Münster 2019 als Faschist bezeichnet werden."

---

### Framework 2: Strukturiert

```
1. [Behauptung zusammenfassen]
2. [Warum falsch/irreführend]
3. [Korrekte Information]
4. [Quelle]
```

**Beispiel:**
"Du sagst, NATO hat Russland ein Versprechen gebrochen.
Das stimmt nicht - es gibt keinen schriftlichen Vertrag.
Gorbatschow selbst sagte 2014, das Thema wurde nie diskutiert.
Quelle: Spiegel Interview 2014."

---

### Framework 3: Sokratisch

```
1. Frage stellen, die Widerspruch aufdeckt
2. Antwort abwarten
3. Nächste logische Frage
4. Bis Widerspruch offensichtlich
```

**Beispiel:**
"Wenn Remigration nicht ethnisch ist, wie identifizierst du 'nicht assimilierte Deutsche'?
Am Nachnamen? An der Hautfarbe?
Was ist das Kriterium?"

---

### Framework 4: Source-Request

```
Interessante Behauptung. Quelle?

[Wenn keine kommt:]
Ohne Quelle ist das eine Meinung, kein Fakt.

[Wenn schlechte Quelle kommt:]
[Analyse der Quelle] → [Bessere Quelle anbieten]
```

---

### Framework 5: Für Wiederholungstäter

```
Du hast diese Behauptung jetzt [X] mal wiederholt.
Ich habe [X] mal widerlegt mit [Quellen].

Entweder du hast ein neues Argument,
oder wir sind hier fertig.
```

## Schnell-Referenz: Häufige Desinformation

| Behauptung | Typ | Quick-Debunk |
|------------|-----|--------------|
| "Flüchtlinge = mehr Vergewaltigung" | Statistik | Gesetzesänderung 2016 |
| "NATO hat versprochen..." | Fake History | Kein Vertrag existiert |
| "Höcke ist kein Faschist" | Denial | OVG Münster 2019 |
| "Remigration ist nicht ethnisch" | Framing | Sellner: "ethnisch-kulturell" |
| "Verfassungsschutz ist politisch" | Verschwörung | 5 Jahre + Gerichte |
| "Beide Seiten gleich schlimm" | False Equiv | 200+ vs <5 Tote |
| "Der Markt regelt Wohnen" | Ideologie | Berlin 90er: 4M→1M |
| "Migranten liegen auf der Tasche" | Wirtschaft | Netto-Einzahler (IAB) |
